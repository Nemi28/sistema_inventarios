# 📦 Sistema de Gestión de Inventarios

Sistema completo de gestión de inventarios desarrollado con tecnologías modernas.

## 🛠️ Tecnologías

- **Backend**: Node.js + Express + MySQL
- **Frontend**: React.js
- **Base de datos**: MySQL

## 📁 Estructura del proyecto
sistema_inventarios/
├── backend/          # API REST con Node.js
│   ├── src/
│   │   ├── config/
│   │   ├── controllers/
│   │   ├── models/
│   │   ├── routes/
│   │   └── app.js
│   └── package.json
├── frontend/         # Aplicación React
│   ├── src/
│   └── package.json
└── README.md

## 🚀 Instalación

### Backend

cd backend
npm install
npm run dev
### Frontend
cd frontend
npm install
npm start
